<b>Usage:</b> imagesorter.py [OPTIONS] SOURCE_DIRECTORY DESTINATION_DIRECTORY<br><br>
<b>Options:</b><br>
  -r, --recursive Recursively photo search<br>
  -m, --move Move sorted photos<br>
